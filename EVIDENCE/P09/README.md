# P09 - SBOM & SCA Evidence

This directory contains artifacts generated by the SBOM and SCA workflow.

## Files

- **sbom.json** - Software Bill of Materials in CycloneDX JSON format (generated by Syft)
- **sca_report.json** - Detailed vulnerability scan report (generated by Grype)
- **sca_summary.md** - Human-readable summary of vulnerabilities by severity

## Generation

These artifacts are automatically generated by the `Security - SBOM & SCA` workflow in `.github/workflows/ci-sbom-sca.yml` and uploaded as GitHub Actions artifacts.

Waivers are managed in `policy/waivers.yml`.
